파이썬다운 코드를 작성하는 방법(Pythonic Code)
===============================================

.. note:: value
    Zen of Python(파이썬의 철학)은 파이썬 프로그래밍 언어의 설계 원칙을 요약한 것입니다.  
    (Tim Peters, "The Zen of Python")

본 문서는 파이썬다운(Pythonic) 코드 작성을 위한 원칙과 예시를 간단히 소개합니다.

.. code :: python
    import this

1. 아름다운것이 보기 흉한 것보다 낫다
2. 명시적인 것이 암시적인 것보다 낫다
3. 단순한 것이 복잡한 것보다 낫다, 복잡한것은 뒤얽힌 것보다 낫다.
4. 펼쳐놓은 수평 구조가 중첩된 계층 구조보다 낫다.  
5. 가독성은 중요하다.
6. 특수한 경우는 규칙을 깨뜨릴 만큼 특별하지 않다.
7. 실용성은 순수성보다 낫다.
8. 오류는 절대 조용히 지나가서는 안된다.
9. 명시적으로 침묵을 허용하는 경우를 제외하고는.
10. 모호함은 피하라.
11. 지금 당장 해야 한다면 지금 당장 하라.
12. 나중에 하는 것보다 낫다.
13. 공간은 시간보다 낫다.
14. 느린 것이 빠른 것보다 낫다.
