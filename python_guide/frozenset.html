frozenset 객체란?

1. 개념

  • frozenset은 파이썬의 내장 불변(immutable) 집합 자료형입니다.
  • set과 거의 동일하지만, 한 번 생성하면 원소를 추가/삭제할 수 없습니다.
  • 불변이기 때문에 set과 달리 딕셔너리의 키, set의 원소로 사용할 수 있습니다.

2. 생성 방법

fs = frozenset([1, 2, 3])
print(fs)  # frozenset({1, 2, 3})

3. 주요 특징

  • set과 마찬가지로 합집합, 교집합, 차집합 등 집합 연산 지원
  • 불변(immutable)이라 원소 추가/삭제 불가
  • hashable(해시 가능)하므로 dict의 키, set의 원소로 사용 가능

4. 주요 메서드 예시

fs1 = frozenset([1, 2, 3])
fs2 = frozenset([3, 4, 5])

print(fs1 | fs2)  # 합집합: frozenset({1, 2, 3, 4, 5})
print(fs1 & fs2)  # 교집합: frozenset({3})
print(fs1 - fs2)  # 차집합: frozenset({1, 2})
print(fs1 ^ fs2)  # 대칭차집합: frozenset({1, 2, 4, 5})

5. set과의 차이점

  • set: 가변(mutable), 원소 추가/삭제 가능
  • frozenset: 불변(immutable), 원소 추가/삭제 불가
s = set([1, 2, 3])
s.add(4)  # 가능
fs = frozenset([1, 2, 3])
# fs.add(4)  # AttributeError 발생

6. 활용 예시

  • 딕셔너리의 키로 사용
  • set의 원소로 사용
my_dict = {frozenset([1, 2]): "값"}
print(my_dict[frozenset([1, 2])])  # "값"

my_set = set()
my_set.add(frozenset([1, 2]))
print(my_set)  # {frozenset({1, 2})}
TIP: frozenset은 데이터의 불변성과 해시 가능성이 필요한 상황에서 매우 유용합니다.
